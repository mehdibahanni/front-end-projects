CSS

Style Sheets (abbreviation: CSS) It is a formatting language for web pages that is concerned with the form and design of websites. It was specifically designed to separate formatting from the content of the written document. This applies to the colors, fonts, images, and backgrounds that are used on the pages, with complete flexibility and ease. This technology is concerned with the overall appearance of website pages, including colors, images, etc. Wikipedia

JavaScript

JavaScript, commonly known as "JS", is a high-level programming language, mainly used in web browsers to create more interactive pages. It is currently being developed by the TC39 committee managed by the ECMA standards organisation. JavaScript began to be used as a programming language aimed at amateur and non-professional programmers. Wikipedia
